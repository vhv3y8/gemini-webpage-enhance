# Gemini Webpage Download Markdown

![](./promo/dropdown-capture.png)

Simple chrome extension to download gemini page chat.

### How to build

```bash
pnpm install
pnpm build:zip
```

### How to apply at chrome

1. Build extension zip
2. Go to `chrome://extensions`
3. Turn on developer mode
4. Drop `gemini-chat-downloader.zip` to the page, or Click "Load unpacked" and Select the `dist` folder
